# 칵테일러

칵테일 검색 및 조회, 칵테일 관련 프로그램 신청을 위한 반응형 웹사이트

## 개발자

- 백시아 : Front(메인 페이지, 헤더, 푸터, 배너, age gate)
- 양현석 : Front(Our Rums 페이지)
- 정재현 : Front(Flavors Range 페이지)
- 최혜수 : Front(Experiences 페이지)

## 기술

`HTML/CSS` `Javascript` `jQuery`

## 특징

- 반응형
- 나이 검사(로컬스토리지 저장)
- 메인 슬라이더
- 향, 종류에 따른 럼 검색
- 과일향 럼 정보 조회
- 칵테일 관련 프로그램 신청
